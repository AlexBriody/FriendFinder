# FriendFinder

A friend finder app that finds you a (star wars) "friend."

## How It Works

* Users take a survey consisting of 10 questions.
* Based on the survey answers, a (star wars) friend is chosen using a compatibility algorithm.

## Built With

* Express
* jQuery
* APIs
* AJAX
* Javascript
* Bootstrap
